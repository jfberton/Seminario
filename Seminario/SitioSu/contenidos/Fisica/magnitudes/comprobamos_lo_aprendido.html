Magnitudes

  1. Las magnitudes son características que pueden ser definidas de forma numérica.  ( JXUwMDBl )
  2. Las magnitudes fundamentales son aquellas que no se pueden definir en función de ninguna otra magnitud.  ( JXUwMDFl )
  3. El tiempo es una magnitud derivada. ( JXUwMDFl )
  4. La medición de magnitudes se realiza por medio de instrumentos calibrados.( JXUwMDFl )

Las siguientes Magnitudes son escalares

  1. Masa ( JXUwMDFl )
  2. Peso  ( JXUwMDBl )
  3. Velocidad  ( JXUwMDFl )
  4. Tiempo  ( JXUwMDFl )

Las siguientes Magnitudes son vectoriales

  1. Superficie  ( JXUwMDFl )
  2. Aceleración ( JXUwMDFl )
  3. Fuerza  ( JXUwMDBl )
  4. Peso  ( JXUwMDFl )
