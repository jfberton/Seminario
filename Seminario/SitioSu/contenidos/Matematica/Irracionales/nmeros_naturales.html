Números Naturales

Los números naturales son aquellos que se usan para contar y numerar. Este conjunto numérico presenta el 1 como primer elemento, pero no tiene último elemento.

La notación que se emplea para identificar el conjunto de números naturales es:

                                               N = {1, 2, 3. … }

 Algunas propiedades importantes son:

  1. N es un conjunto discreto porque entre dos números naturales siempre hay un número finito de números naturales.

  2. Todo número natural a, tiene su sucesor 1 + a.

  3. Tanto la suma como el producto de números naturales es un número natural, en cambio no sucede lo mismo con la resta y la división.

  4. Un número natural se puede expresar como producto de otros números naturales, que se llaman factores o divisores del primero.

             Ejemplo: 120 = 23. 3. 5
